What is KubeClaw?

KubeClaw is a bridge between the OpenClaw agentic framework and Kubernetes. It allows you to run distributed node clusters that act as the eyes, ears, and hands of your AI agents.

Deploy in seconds

# Apply the KubeClaw deployment
kubectl apply -f https://raw.githubusercontent.com/jianan1104/kubeclaw/main/k8s/deployment.yaml

Why KubeClaw?

  • Native Scaling: Use kubectl scale to manage your AI's workers.
  • Safe Sandboxing: Every node is a fresh container.
  • Resource Management: Limit CPU/Memory for your agent's tasks.
